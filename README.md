# Premium SaaS Portfolio

A modern full-stack personal portfolio built with React, Tailwind CSS, Framer Motion, Express, MongoDB, and Nodemailer.

## Project Structure

- `/client` — React + Vite frontend
- `/server` — Express backend with MongoDB

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy environment files:
   ```bash
   cp server/.env.example server/.env
   ```

3. Start app:
   ```bash
   npm run dev
   ```

## Backend APIs

- `POST /api/contact`
- `GET /api/resume-download-count`
- `POST /api/resume-download`

## Philosophy

Built for premium SaaS feel, smooth interactions, and full-stack contact tracking.
